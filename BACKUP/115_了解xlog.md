# [了解xlog](https://github.com/myogg/Gitblog/issues/115)

很早就看到过xlog，但是一看到这个又要用钱包 又要用区域链 自己从来就没有用过去区域链 钱包 而且也不会创建 ，也没有敢尝试 。

前两天也是抱着试试的心态，不知道怎么就开通了 ，说实话 这个怎么登录的我都不知道 反正是能发文章了 ，仔细看了一下 嗯 还是非常方便的 就是有些验证的麻烦了一些 。

先用着吧 我觉得这个唯一的好处就是 上传一些图片视频你就不用考虑到容量的问题，你不用再去找各种各样的网盘，各种各样的视频盘 ，你可以在文章中直接上传图片视频，而且感觉速度还挺快，包括自己介绍说 ，这个是 存储在区域链之内的，就是每个cdn都可能存着你的文章呀，所以你什么也丢不了，

勇于尝试，敢于尝试 。让我们拭目以待吧

xLog是一个现代化的开源博客系统，使用区块链技术存储博客数据，包括配置、文章、评论等。它具有丰富的交互功能，可以关注博客、评论、点赞文章，并将文章铸造为NFT。

xLog基于proselog项目开发，使用Next.js + Tailwind CSS + TypeScript + TanStack Query构建，具有完善的开发体验。它使用Crossbell区块链进行社交活动，无需购买gas费用。

每个博客都是一个属于用户的NFT，配置和文章存储在NFT中。然而，xLog的使用需要安装以太坊钱包，这对于没有接触过区块链的普通用户可能是一个负担。
