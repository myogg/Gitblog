# [ClouDNS 免費二級域名](https://github.com/myogg/Gitblog/issues/109)

它的优势包括：

• 永久免费 —— 提供 Free DNS Plan 和免费二级域名

• 解析稳定快速 —— 全球 Anycast 网络

• 支持 DDNS —— 非常适合 NAS / 家庭宽带公网 / Cloudflare Tunnel

• 界面清晰易用 —— 新手完全无压力

• 不会乱封域名 —— 稳定性远胜 Freenom

![](https://pic.imgdd.cc/item/693104337fc170517f43968c.jpg)

拿到域名后，可以像正常域名一样使用：

• A 记录：指向你的公网 IP

• CNAME：用于 Tunnel 或反代服务

• AAAA：支持 IPv6

• MX：甚至可以做邮箱（不过免费版功能有限）

[www.cloudna.com](www.cloudns.com)
