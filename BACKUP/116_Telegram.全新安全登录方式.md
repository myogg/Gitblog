# [Telegram 全新安全登录方式](https://github.com/myogg/Gitblog/issues/116)

通行密钥（Passkeys）：Telegram 全新安全登录方式

通行密钥（Passkeys）是一种免密码的安全登录技术，它依托公钥私钥加密体系与设备本地验证机制，替代了传统密码形式，能显著提升账户登录的安全性与操作便捷性。

Telegram 已正式上线通行密钥登录功能，该功能可完全取代短信验证码登录。一方面解决了此前国际短信验证码存在的发送成本高、部分地区接收失败等痛点，另一方面也有效规避了短信验证码被劫持的风险，实现了登录安全与便捷性的双重升级。

值得注意的是，通行密钥会存储在用户自主选择的密码管理器中，其安全性与可用性不受 Telegram 平台服务本身的影响，保障了密钥的独立性与稳定性。

核心优势
   * 替代短信验证，无接收地域限制、无劫持风险
   * 依托公钥加密体系，登录安全等级更高
   * 密钥存于个人密码管理器，与平台服务解耦

已支持的客户端：
   * iOS 官方客户端版本 ≥ v12.2.3
   * Android 官方客户端版本 ≥ v12.2.8
   * Desktop 官方客户端和第三方客户端版本 ≥ v6.3.6
   * iOS 和 Android 第三方客户端不支持通行密钥，这是 Telegram 的限制

开启方式：
   * 中文界面：设置 -> 隐私 -> 通行密钥
   * 英文界面：Settings -> Privacy and Security -> Passkey
   * iOS 上无法开启的解决办法：连续点击10次“设置” -> 进入“Debug”界面 -> 点击“Clear Database and Cache” -> 再去开启

温馨提示：为保障账号安全，强烈建议所有符合条件的用户尽快开启通行密钥功能！

尽管 Telegram 已上线通行密钥登录功能，但账号注册环节对手机号码的硬性要求始终未变，无论过去还是现在，手机号都是注册的必备条件。

通行密钥仅优化了 Telegram 的登录体验，并未改变其注册规则 —— 和以往一样，手机号码至今仍是账号注册的必要前提，缺一不可。
