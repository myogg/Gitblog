# [闲聊](https://github.com/myogg/Gitblog/issues/154)

![](https://pic.imgdd.cc/item/694b4eb8363f7cc0e05346dc.png)


昨天隔壁大姐过来聊天。就是说这精神病跟抑郁症有什么区别？我想一下，精神病人脑子不正常。抑郁症是钻了牛角尖，但脑子是正常的，可能是是这一阵他身边自杀的有。

后来大姐走了以后，我又想了一下。这两种病都是活在自己的世界里。我的解释也不是很准确。我就去查了一下。


我简略了一下AI的回答。

一、精神病是一组以脱离现实为核心的症状，主要表现为幻觉与妄想，并非独立疾病。

二、抑郁症是一种以持久情绪低落、兴趣丧失为核心特征的心境障碍疾病。

三、区别在于：精神病是症状，表现为现实扭曲；抑郁症是疾病，核心是情绪衰竭。

四、相同之处在于：两者都可能严重损害社会功能，且重度抑郁症可包含精神病性症状，需要专业治疗。


以前发的那么多，无聊的都是为了测试，今后要多多记录自己的生活。
